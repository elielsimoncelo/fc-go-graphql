# fc-go-graphql

## Instalação

> Instalar os pacotes

- go mod tidy

> Criar a estrutura do graphql

- go run github.com/99designs/gqlgen init

## Atualização

> Gerando novamente a estrutura da aplicacao apos mudancas

- go run github.com/99designs/gqlgen generate

## Executando

> Executar o servidor

- go run ./cmd/server/server.go
